이미지 기본 구조

HTML에서 이미지는 <img> 태그를 사용하여 웹 페이지에 삽입합니다. <img> 태그는 빈 요소로, 닫는 태그가 없습니다.

기본 이미지 예제

<img src="https://via.placeholder.com/300x200" alt="예제 이미지">

이미지 태그 필수 속성

  • src: 이미지 파일의 경로 (필수)
  • alt: 이미지를 표시할 수 없을 때 대체 텍스트 (접근성을 위해 필수)

이미지 경로 예제

절대 경로

<img src="https://example.com/images/photo.jpg" alt="예제 사진">

상대 경로


<img src="photo.jpg" alt="예제 사진">


<img src="images/photo.jpg" alt="예제 사진">


<img src="../images/photo.jpg" alt="예제 사진">

이미지 크기 속성

HTML에서 이미지의 크기를 지정하는 방법은 두 가지가 있습니다:

1. HTML 속성 사용

<img src="https://via.placeholder.com/300x200" alt="HTML 속성으로 크기 조절된 이미지" width="200" height="100">

2. CSS 속성 사용 (더 권장됨)

<img src="https://via.placeholder.com/300x200" alt="CSS로 크기 조절된 이미지" style="width: 200px; height: 100px;">

반응형 이미지

반응형 이미지는 다양한 화면 크기에 맞게 자동으로 크기가 조절됩니다.

CSS를 사용한 반응형 이미지

<img src="https://via.placeholder.com/800x400" alt="반응형 이미지" class="responsive-img">

/* CSS */
.responsive-img {
    max-width: 100%;
    height: auto;
}

figure와 figcaption

<figure>와 <figcaption> 요소를 사용하여 이미지에 캡션을 추가할 수 있습니다. 

<figure>
    <img src="landscape.jpg" alt="자연 풍경">
    <figcaption>아름다운 자연 풍경의 모습</figcaption>
</figure>
