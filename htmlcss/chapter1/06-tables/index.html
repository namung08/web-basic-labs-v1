기본 테이블 구조

HTML 테이블은 다음 태그들로 구성됩니다:

  • <table>: 테이블을 정의
  • <tr>: 테이블 행(row)을 정의
  • <th>: 테이블 헤더 셀을 정의
  • <td>: 테이블 데이터 셀을 정의

테이블 구조를 명확히 하는 태그들:

  • <caption>: 테이블의 제목이나 설명을 추가
  • <thead>: 테이블의 헤더 부분을 그룹화
  • <tbody>: 테이블의 본문 부분을 그룹화
  • <tfoot>: 테이블의 바닥글 부분을 그룹화
