폼의 기본 구조

HTML 폼은 사용자로부터 입력을 받기 위한 요소입니다. 사용자가 입력한 데이터를 수집하여 서버로 전송하거나 클라이언트 측에서 처리할 수 있습니다.

기본 폼 구조

<form action="process.php" method="post">
    <!-- 입력 요소들 -->
    <input type="submit" value="제출">
</form>

주요 속성:

  • action: 폼 데이터를 처리할 서버의 URL
  • method: 데이터 전송 방식 (get 또는 post)
  • name: 폼의 이름

입력 요소

HTML은 다양한 유형의 입력 요소를 제공합니다.

텍스트 입력

<label for="text-input">텍스트 입력:</label>
<input type="text" id="text-input" name="text-input" placeholder="텍스트를 입력하세요">

비밀번호 입력

<label for="password-input">비밀번호:</label>
<input type="password" id="password-input" name="password-input" placeholder="비밀번호를 입력하세요">

<input type="email" id="email-input" name="email-input" placeholder="이메일을 입력하세요">

HTML5 폼 속성

속성 설명 예제
required 필수 입력 필드 지정 <input type="text" required>
placeholder 입력 필드에 표시할 힌트 <input type="text" placeholder="이름 입력">
pattern 입력 값이 일치해야 하는 정규식 패턴 <input type="text" pattern="[A-Za-z]{3}">
min, max 숫자 입력의 최소/최대 값 <input type="number" min="0" max="100">
step 숫자 입력의 간격 <input type="number" step="0.5">
autofocus 페이지가 로드될 때 자동으로 포커스를 받음 <input type="text" autofocus>
autocomplete 자동 완성 기능 켜기/끄기 <input type="text" autocomplete="off">
