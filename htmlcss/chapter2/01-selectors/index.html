선택자란?

CSS 선택자는 스타일을 적용할 HTML 요소를 선택하는 패턴입니다. 선택자를 사용하여 특정 요소를 대상으로 스타일을 적용할 수 있습니다.

기본 선택자

1. 전체 선택자 (*)

이 텍스트는 전체 선택자의 영향을 받습니다.

이 텍스트도 전체 선택자의 영향을 받습니다. 
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

2. 요소 선택자 (태그 선택자)

이 단락은 p 선택자의 영향을 받습니다.

이 텍스트는 span 선택자의 영향을 받습니다. 
p {
    color: blue;
}
span {
    font-weight: bold;
}

3. 클래스 선택자 (.class)

이 단락은 .highlight 클래스 선택자의 영향을 받습니다.

이 단락은 클래스 선택자의 영향을 받지 않습니다.

이 텍스트도 .highlight 클래스 선택자의 영향을 받습니다. 
.highlight {
    background-color: yellow;
    padding: 5px;
}

4. ID 선택자 (#id)

이 단락은 #unique ID 선택자의 영향을 받습니다.

이 단락은 ID 선택자의 영향을 받지 않습니다.

#unique {
    color: red;
    font-weight: bold;
}

결합 선택자

1. 자식 선택자 (>)

이 단락은 div.parent > p 선택자의 영향을 받습니다.

이 단락은 영향을 받지 않습니다 (직접적인 자식이 아님).

.parent > p {
    border-left: 3px solid blue;
    padding-left: 10px;
}

2. 자손 선택자 (공백)

이 단락은 div.ancestor p 선택자의 영향을 받습니다.

이 단락도 영향을 받습니다 (직접적인 자식이 아니더라도 자손임). 

.ancestor p {
    color: green;
}

3. 인접 형제 선택자 (+)

제목

이 단락은 h4 + p 선택자의 영향을 받습니다 (h4 바로 다음에 오는 p).

이 단락은 영향을 받지 않습니다 (인접 형제가 아님).

h4 + p {
    font-style: italic;
}

4. 일반 형제 선택자 (~)

제목

이 단락은 h4 ~ p 선택자의 영향을 받습니다 (h4 이후의 모든 p).

중간에 다른 요소가 있어도

이 단락도 영향을 받습니다 (같은 부모 아래의 형제).

h4 ~ p {
    text-decoration: underline;
}

가상 선택자

1. 가상 클래스 (:)

마우스를 올려보세요
  • 첫 번째 항목 (:first-child)
  • 두 번째 항목
  • 세 번째 항목
  • 마지막 항목 (:last-child)

이 단락에는 가상 요소도 사용됩니다. 

a.hover-example:hover {
    color: red;
    text-decoration: none;
}
li:first-child {
    font-weight: bold;
}
li:last-child {
    color: blue;
}

2. 가상 요소 (::)

이 단락의 첫 번째 줄은 ::first-line 가상 요소의 영향을 받습니다. 텍스트를 충분히 길게 작성하여 여러 줄에 걸쳐 표시되도록 합니다. 브라우저 창의 크기에 따라 첫 번째 줄이 결정됩니다.

이 단락의 첫 글자는 ::first-letter 가상 요소의 영향을 받습니다. 

p.first-line::first-line {
    font-weight: bold;
}
p.first-letter::first-letter {
    font-size: 2em;
    color: red;
}

속성 선택자

속성 선택자 사용 예

외부 링크 (target 속성 있음)
내부 링크
a[target] {
    color: orange;  /* target 속성이 있는 링크 */
}
input[type="email"] {
    border-color: green;  /* type이 email인 입력 필드 */
}
