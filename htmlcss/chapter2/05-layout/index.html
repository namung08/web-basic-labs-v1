기본 레이아웃 속성

display 속성

display: inline inline 요소는 콘텐츠만큼만 공간을 차지합니다.
display: block
block 요소는 가로 전체 공간을 차지합니다.
display: inline-block inline과 block의 특성을 모두 가집니다.

position 속성

position: static (기본값)
position: relative
position: absolute
position: fixed

float 속성

float: left
float: right
clear: both
